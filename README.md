# 🛒 Sales Management System

Meu ultimo trabalho, depois de passar meses longe dos codigos

Um sistema de gerenciamento de vendas e estoque desenvolvido em **Python**, com foco em organização, produtividade e boas práticas de desenvolvimento.

O projeto foi criado para facilitar o controle de produtos, estoque e vendas, oferecendo uma interface simples e intuitiva para pequenas empresas e comércios.

---

## 🚀 Tecnologias Utilizadas

- Python
- SQLite
- CustomTkinter
- Tkinter
- Programação Orientada a Objetos (POO)
- SQL
- Git & GitHub

---

## ✨ Funcionalidades

- 📦 Cadastro de produtos
- 📊 Controle de estoque
- 💰 Registro de vendas
- 🧾 Histórico de vendas
- 🔍 Pesquisa de produtos
- ✏️ Atualização de produtos
- 🗑️ Exclusão de produtos
- 📉 Atualização automática do estoque após uma venda
- 💾 Persistência dos dados utilizando SQLite
